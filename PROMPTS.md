# Prompt Templates

## Architect Review
Use when you want a full state assessment before making decisions.

> "Act as the architect. Review the current state of [FOCUS AREA].
> Give me:
> 1. Current state assessment
> 2. Risks and blockers 🟡
> 3. Recommended next 3 steps with rationale
> 4. Open questions ❓"

## Sprint Planning
Use when starting a new sprint.

> "Act as the architect. The client wants [GOAL].
> Break this into tasks with complexity estimates (S/M/L).
> Include acceptance criteria and dependencies for each task.
> Feature-boxed sprint — ends when all tasks are done."

## Client Decision
Use when the client needs to make a choice.

> "Client decision needed on [TOPIC].
> Option A: [pros/cons/effort]
> Option B: [pros/cons/effort]
> My recommendation: [X] because..."

## Change Request
Use when the client asks for something outside current scope.

> "Change request: [REQUEST].
> Impact analysis:
> - Effort:
> - Timeline shift:
> - Dependencies:
> - Alternatives:
> My recommendation:"

## Session Summary
Auto-generated at the end of each session.

> "Session summary:
> - What was done:
> - Decisions made:
> - Files changed:
> - Current sprint progress: [%]
> - Blockers:
> - Next session priorities:"
